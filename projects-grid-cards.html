Recent Projects


Artificial Intelligence Derived Predictive Analysis of COVID-19 over Time (AID PACT)

This project was developed during the 48-hour NASA Space App Challenge focused on COVID-19. We developed a website which shows an interactive map with cases of COVID-19 globally. This interactive map was integrated with a Bayesian Ridge Regression Model, which was used to predict future COVID-19 cases for each country

BayHax

Won 2nd place internationally at the MLH NotUniversity Hacks, which had over 300 participants. Created BayHax, which is an emotional health tracking teddy bear outfitted with a Raspberry Pi, Raspberry Pi Camera, speaker, and buttons. BayHax was connected to a website using python fetch requests. This website was used to analyze and display the emotional health data collected by BayHax.

Image Editor

As a pinnacle project for the Java track, the Image Editor required every aspect of the basic Java language. It could input, output, edit, apply custom filters, and implemented undo/redo functionality.
